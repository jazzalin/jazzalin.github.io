Gnuplot GUI (July 2016 - Present)

In order to facilitate the representation of data measurements in a laboratory setting, I decided to come up with a simple GUI which would use the powerful plotting software gnuplot (the use of gnuplot can prove to be a little cumbersome for some ...). The interface itself would be coded with the scripting language Tcl and its GUI toolkit Tk. The use of Tcl/Tk would make it easy to develop a cross-platform desktop application. My intention is to package the interface with a number of statistical analysis features which I am currently learning through a stats class (MTE 201) and on my own time. One of them is a simple best-fitting-line subroutine I implemented in Fortran 90 which uses gradient descent to perform the regression analysis.

Chaos theory and the double pendulum (2014 - Present)

Ever since reading about this mathematical theory in James Gleick's Chaos: Making a New Science, I've been intrigued by the manifestation of 'chaos' in the world (remember when Jeff Goldblum played Dr. Ian Malcolm, the chaos theorist, in Jurassic Park? Same idea). A system which is well known to showcase chaos is the double pendulum. By contrast to the simple pendulum and determinism, the double pendulum introduces what is called 'deterministic chaos', which essentially challenges our understanding of reality. After writing an essay about it in Grade 11, I decided to build both a simulated model and a physical one. I intend to numerically simulate the differential equations with Fortran, while laser cutting the various parts needed to put the pendulum together.

NXT plotter (October - November 2015)

My end of term group project consisted in a LEGO NXT robot affixed to a whiteboard (of our own making), along which it travelled horizontally and vertically to plot polynomial functions. The functions, which were kept basic... at first, were input by the user on a computer running a plotting assistant. I developed this interface using the Qt framework and was responsible for the robot's input functions. The entire system proved to be quite effective at interpreting the coefficients input by the user and plotting the corresponding function on the whiteboard.
